## Setup

1. Clonar el proyecto:
Parado sobre el directorio donde querés almacenar el proyecto, ejecutá
```
git clone https://github.com/franncoronel/g5-backend.git
```
2. Instalar las dependencias:
```
npm i
```
3. Con las dependencias instaladas, ejecutar el proyecto:
```
npm run dev
```
